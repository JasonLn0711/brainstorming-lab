## Bridge Rule

Planning keeps the map. Brainstorming keeps the detailed thinking.

Planning note shape:

```markdown
- Idea:
  - Summary:
  - Canonical brainstorm:
  - Next test:
  - Capacity impact: none / maybe / yes
```

## When To Update Planning

Update planning only when:

- the idea affects today's or this week's capacity
- the idea becomes a real project
- the next action belongs on an agenda
- a durable lesson should be promoted
- a project repo is created, paused, or closed

## When Not To Update Planning

Do not update planning for every new paragraph, objection, option, or raw thought here.

Planning should stay calm enough to guide the day.
