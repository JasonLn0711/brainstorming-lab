# Personal Admin Records

This folder stores redacted, git-trackable summaries for personal administrative events that need more structure than a day-note line.

Rules:

- Keep raw PDFs, spreadsheets, Gmail exports, identity data, bank-account records, labor-insurance records, recommendation letters, and similar private source files outside git.
- Put raw files in `~/system-hub/admin/inventory/` with owner-only permissions.
- Track only retrieval-safe summaries here: timeline, status, deadlines, source locator, missing evidence, and next actions.
- Planning receives only the short bridge note when the record affects today, this week, or an upcoming deadline.
