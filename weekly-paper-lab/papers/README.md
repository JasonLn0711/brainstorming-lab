# Paper Records

Selected weekly papers live here as one folder per paper:

```text
<YYYY-Www>-<slug>/
```

Start from `weekly-paper-lab/templates/`. Keep `paper.yaml` as the canonical
record, and keep the Markdown files as the human-readable lab notebook.
`07_scoring_report.md` explains why the selection, Idea OS connection, and
synthesis evidence were assigned. `08_scientific_evaluation.md` explains the
post-reading scientific-quality score, which is recomputed from
`scientific_evaluation.yaml`.

Do not store model weights, datasets, caches, private evidence, or generated
large outputs in this directory.
